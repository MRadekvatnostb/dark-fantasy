# Тень Бездны

Текстовая RPG с системой боя, прокачкой и сохранением прогресса.

![Java](https://img.shields.io/badge/Java-21-orange)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-16-blue)

## О проекте

**"Тень Бездны"** — атмосферная текстовая ролевая игра, в которой ты выбираешь одного из пяти уникальных классов и отправляешься в опасное путешествие по мрачным локациям.

### Возможности

- **5 playable классов**:
    - Рыцарь
    - Вампир
    - Лучник
    - Маг
    - Нежить

- **Сложная боевая система**:
    - Критические удары
    - Вампиризм
    - Эффекты (стан, кровотечение, яд и др.)
    - Ультимативные способности у каждого класса
    - Система энергии

- **Инвентарь** с расходниками и артефактами, дающими временные баффы

- **4 локации** + финальный босс

- **Сохранение прогресса** в PostgreSQL (можно продолжить после выхода)

- Атмосферная подача текста (постепенный вывод, разделители)

## Как запустить

1. Установи **PostgreSQL** и создай базу `Dark_Base`
2. Создай пользователя `postgres` с паролем `1337` (или измени в `application.properties`)
